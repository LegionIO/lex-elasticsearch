# lex-elasticsearch

Elasticsearch integration for [LegionIO](https://github.com/LegionIO/LegionIO). Provides indexing and search operations against Elasticsearch clusters.

## Installation

```bash
gem install lex-elasticsearch
```

## Requirements

- Ruby >= 3.4
- [LegionIO](https://github.com/LegionIO/LegionIO) framework
- Elasticsearch cluster
